Claude Code Skill

slide-craft

将任何主题一键生成精美的自包含 HTML 演示文稿，零依赖，浏览器直接打开演示。

外部依赖项 零依赖
内置主题风格 4 套主题
语言支持 中 / EN

工作流程

六步工作流

①
Gather

从用户消息提取主题、页数、主题风格等参数，缺省时使用默认值。

②
Draft Plan

生成结构化计划文件（.md），每页含布局类型、关键信息与演讲注释。

③
Confirm

暂停等待用户确认，支持修改后重新确认。未经确认不生成 HTML。

④
Select Theme

读取 design_tokens.md，选定匹配主题的 CSS 变量集合备用。

⑤
Generate

读取 base_template + layouts + design_guidelines，逐页构建 HTML。

⑥
Deliver

写入 <topic-slug>-slides.html，单文件自包含，浏览器直接打开。

设计系统

7 种布局 × 4 套主题

布局库

  • slide-grid-cover — 封面标题页
  • card-stack — 并列要点卡片
  • timeline-grid — 流程 / 步骤
  • relationship-grid — 2×2 矩阵
  • two-column — 对比分析
  • slide-cover-center — 章节过渡
  • slide-grid — 文字 + 支撑卡片

主题系统

  • warm — 人文暖色，学术风格
  • tech — 深色科技，开发者风格
  • minimal — 商务简洁，纯白底色
  • forest — 自然森林，深绿基调

快速上手

立即开始使用

安装 plugin 后，在 Claude Code 中输入 /slide-craft 命令即可启动。

  • 将 skill 文件放入 ~/.claude/plugins/slide-craft/skills/slide-craft/
  • 调用：/slide-craft [主题] [--pages N] [--theme warm|tech|minimal|forest]
  • 输出：当前目录下的 <topic-slug>-slides.html，浏览器直接打开演示
